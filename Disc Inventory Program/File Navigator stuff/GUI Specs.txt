The GUI Specs

The GUI needs:
--two buttons
--a vertical line splitting the window
--a textbox
--a box for displaying information(this will need to be quite wide, about 600 pixels at a minimum)

--color choice is yours. Make it awesome

-button “Select Origin Directory”
	--this button saves the selected item in the ListBox to an object File(you can also use String if you want to called parentDirectory
		--also enables "Inventory" so it can be clicked
	--if there is nothing selected, present an error message
		//the code for the ListBox content is mine, it's quite complicated

-button “Inventory” (default as disabled)
	--this button checks if a textbox has a filename in it, so long as the textbox is not empty, it will do stuff
		--if the textbox is not empty(if(textbox != "")) then call inventory(parentDirectory) and create a .csv file using the filename in the textbox
			--inventory(parentDirectory) is the class I'm making with recursion funkyness

--There needs to be a void method called printInfo(String info)
	--this method does two things: first, if writes 'info' to a .csv file you created need to create in the 'Inventory' Button
		--make sure you append, do not truncate(erase and write)
	--Then, split the string using commas(",") and write the String array to the display(on one line) putting a newline at the end
		--The first String must be no more than 30 characters, then 3 spaces, then the next string(4 characters) then 3 spaces. The other strings are all seperated by spaces, but no length limits
			--make sure the writing method can cope with any length string array
		--An example of output is below, make sure the spacing is correct. 

--There needs to be a void method called displayList(String [] list)
	--it takes a String[] called list and puts it into the List Box
